# workshop-frontend-testing

## Project setup

- [git](https://git-scm.com/)
- [Node](https://nodejs.dev/)
- [pnpm](https://pnpm.io/): `npm install -g pnpm`
- `git clone https://github.com/gautemo/workshop-frontend-testing.git`
- `cd workshop-frontend-testing`
- `pnpm i`

## Run frontend

- `pnpm dev`
